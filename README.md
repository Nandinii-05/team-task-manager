# 🚀 Team Task Manager (Full Stack Backend)

A Task Management Web Application built using Node.js, Express, MongoDB, and EJS.  
This application allows users to manage projects, tasks, and authentication with role-based access.

---

## 🌐 Live Demo
👉 https://team-task-manager-production-1996.up.railway.app

---

## 📌 Features

- 🔐 User Authentication (Login/Register)
- 🧑‍🤝‍🧑 Role-Based Authorization
- 📁 Project Management (CRUD)
- ✅ Task Management (Create, Update, Delete)
- ⭐ Task Status Tracking
- 📊 Dashboard View
- ☁️ Cloud Deployment using Railway
- 🗄️ MongoDB Atlas Database Integration

---

## 🛠️ Tech Stack

- **Backend:** Node.js, Express.js  
- **Database:** MongoDB Atlas  
- **Templating Engine:** EJS  
- **Authentication:** JWT  
- **Deployment:** Railway
